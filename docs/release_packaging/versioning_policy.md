# Versioning Policy

Early public template versioning can start at:

```text
v0.1.0-alpha.1
```

## Version Format

Use:

```text
vMAJOR.MINOR.PATCH-prerelease.N
```

Example:

```text
v0.1.0-alpha.1
```

## Asset Naming

Use:

```text
codex-native-closed-loop-<version>.zip
```

Recommended first asset name:

```text
codex-native-closed-loop-v0.1.0-alpha.1.zip
```

## Meaning

A version does not imply runtime readiness. An alpha version means
template/readiness stage.

Publicization and release upload are separate approvals. Tag creation, GitHub
Release creation, release asset upload, OpenAI application submission, commit,
and push each require separate owner approval.
